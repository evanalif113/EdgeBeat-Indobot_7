Projek Akhir WEMOS B Indobot 7

Aplikasi Pemantauan Kesehatan Berbasis Sensor Digital

Dengan memanfaatkan sensor optik pada module MAX30102 yang digunakan untuk memantau denyut nadi dan saturasi oksigen dalam darah. Sensor ini menggunakan teknologi PPG (Photoplethysmography) untuk mengukur perubahan volume darah di pembuluh darah.

Fitur Utama

Monitoring Denyut Nadi

Memantau irama denyut nadi dengan memanfaatkan teknologi PPG (Photoplethysmography).

Kadar Oksigen

memantau saturasi oksigen dalam darah untuk menjaga kesehatan yang optimal.

Riwayat Data

Menyimpan data rekam pengguna dalam database.
